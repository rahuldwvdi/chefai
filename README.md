# ChefAI 🍳

ChefAI is an AI-powered cooking assistant that generates recipes based on ingredients available in your kitchen.

## Features
- Ingredient-based recipe suggestions
- AI-generated recipes using LLM
- Clean Streamlit web interface
- Deployed online

## Tech Stack
- Python
- Streamlit
- Groq LLM API
- Pandas

## Live Demo
https://your-streamlit-app-link

## How It Works
1. User enters ingredients
2. System matches recipes from database
3. AI generates custom recipes
4. Results displayed in UI
